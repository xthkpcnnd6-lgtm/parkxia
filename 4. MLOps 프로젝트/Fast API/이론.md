# 📘 FastAPI 기초 이론 가이드

> **핵심 요약:** FastAPI는 파이썬 3.8+ 기반의 초고속 웹 프레임워크로, 
> 특히 **머신러닝 모델의 서빙(Serving)**을 위해 설계된 현대적인 도구입니다.

---

## 1. FastAPI란?
FastAPI는 파이썬의 표준 **타입 힌트(Type Hints)**를 사용하여 API를 구축하는 도구입니다. 
단순히 웹 서버를 만드는 것을 넘어, 데이터의 유효성을 검증하고 자동으로 문서를 만들어주는 영리한 프레임워크입니다.

### 🌟 주요 장점
*   **압도적 성능:** NodeJS, Go에 필적하는 속도 (Starlette 및 Pydantic 기반)
*   **개발 생산성:** 직관적인 문법으로 버그 발생률을 낮추고 개발 속도 향상
*   **자동 문서화:** 코드를 작성하는 즉시 실시간 API 명세서 생성 (`Swagger UI`)
*   **안정성:** 데이터 타입 검증을 통해 런타임 에러 최소화

---

## 2. 3대 핵심 기술 스택
FastAPI의 성능을 지탱하는 삼총사입니다. 

| 요소 | 역할 | 비유 |
| :--- | :--- | :--- |
| **Pydantic** | 데이터 검증 및 직렬화 (JSON 변환) | 재료의 신선도를 검사하는 품질 검수원 |
| **Starlette** | 라우팅 등 웹 기능의 기초 골격 | 공장의 기둥과 전력망 같은 기본 인프라 |
| **Uvicorn** | 비동기 처리를 지원하는 서버 엔진 (ASGI) | 공장을 돌리는 초고속 모터 |

---

## 3. 핵심 동작 원리 및 용어

### ① ASGI (Asynchronous Server Gateway Interface)
기존의 동기식 방식(WSGI)과 달리, **비동기(Async)** 처리를 지원합니다. 한 명의 손님이 주문하고 음식을 기다리는 동안 다음 손님의 주문을 받을 수 있는 효율적인 시스템입니다.

### ② 타입 힌팅 (Type Hinting)
