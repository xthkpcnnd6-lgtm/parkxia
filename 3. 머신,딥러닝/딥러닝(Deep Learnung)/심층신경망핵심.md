# 📑 Module 7. 심층신경망(DNN) 핵심 원리 정리

## 1. DNN의 기본 연산 구조
모든 층은 **선형 결합**과 **활성화 함수**의 반복으로 이루어집니다.
* **선형 결합 (Linear):** $z = XW^T + b$ (데이터에 가중치를 곱하고 편향을 더함)
* **활성화 (Activation):** $a = \text{activation}(z)$ (비선형성을 추가하여 복잡한 패턴 학습 가능)

---

## 2. 분류 모델의 3단계 흐름
모델이 정답을 맞히기까지는 다음의 세 단계를 거칩니다.

### ① 로짓 (Logits)
* 모델의 최종 출력층에서 나온 **날것의 점수**.
* 범위 제한이 없고 합이 1일 필요도 없음. (예: [2.0, 1.0, 0.1])

### ② 소프트맥스 (Softmax) : 확률화
* 로짓을 **0~1 사이의 확률**로 변환.
* 모든 클래스 확률의 **총합은 항상 1**.
* 큰 점수는 더 크게, 작은 점수는 더 작게 만들어 차이를 강조함.

### ③ 교차 엔트로피 (Cross-Entropy) : 손실 계산
* 모델이 예측한 확률($p_t$)이 정답에 가까울수록 손실(Loss)은 0에 수렴.
* 틀릴수록 손실값은 기하급수적으로 커짐 ($L = -\log p_t$).

---

## 3. 역전파(Backpropagation)의 마법
복잡한 미분 과정을 거치면, 결국 가중치를 업데이트하기 위한 최종 계산식은 놀라울 정도로 단순해집니다.

> **로짓에 대한 미분값 = 예측 확률($p_k$) - 실제 정답($y$)**

* **정답 클래스 ($k=t$):** $p_t - 1$ (예측이 100%가 아니면 음수값 $\rightarrow$ 점수를 높이는 방향으로 학습)
* **오답 클래스 ($k \ne t$):** $p_k - 0$ (예측이 0%가 아니면 양수값 $\rightarrow$ 점수를 낮추는 방향으로 학습)
* **결론:** 이 단순한 **'차이'**가 신경망을 타고 역방향으로 흐르며 모든 가중치를 수정합니다.

---

## 4. 실전 도구 선택 가이드

| 구분 | 다중 분류 (Multi-class) | 멀티 라벨 (Multi-label) |
| :--- | :--- | :--- |
| **설명** | 여러 개 중 **정답 1개** | 정답이 **여러 개 가능** |
| **활성화 함수** | **Softmax** (클래스 간 경쟁) | **Sigmoid** (각자 독립 판단) |
| **손실 함수** | `CrossEntropyLoss` | `BCEWithLogitsLoss` |

## 5. 데이터 불균형(Rare Class) 대응 전략
데이터가 극도로 부족한 '희귀 클래스'가 있을 때 사용하는 기법입니다.
1. **Weighted Sampler:** 학습 시 희귀 데이터를 더 자주 뽑아서 모델에게 많이 보여줌.
2. **Loss Weighting:** 희귀 데이터를 틀렸을 때 벌점(Loss)을 더 크게 부여함 (`pos_weight`).
3. **Focal Loss:** 맞히기 쉬운 데이터보다 어려운(희귀한) 데이터에 집중해서 학습함.

    # 5. 가중치 업데이트 (Step)
