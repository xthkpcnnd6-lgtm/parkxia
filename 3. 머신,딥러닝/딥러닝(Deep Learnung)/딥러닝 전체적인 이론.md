# 🧠 딥러닝 (Deep Learning) 핵심 정리

---

## 1️⃣ 딥러닝이란?

> 👉 인공신경망(Neural Network)을 **깊게 쌓아 복잡한 패턴을 학습하는 기술**

* 머신러닝의 한 분야
* 이미지, 음성, 자연어 처리에서 강력한 성능

---

## 2️⃣ 기본 구조

```
입력층 (Input Layer)
    ↓
은닉층 (Hidden Layers)
    ↓
출력층 (Output Layer)
```

* 층(Layer)이 많을수록 “Deep”
* 각 층은 뉴런(Node)으로 구성됨

---

## 3️⃣ 핵심 구성 요소

### ✔ 뉴런 (Neuron)

* 입력값 × 가중치 + 편향 → 출력

---

### ✔ 가중치 (Weight) & 편향 (Bias)

* 모델이 학습하는 값
* 입력의 중요도를 조절

---

### ✔ 활성화 함수 (Activation Function)

> 👉 비선형성 추가 (중요)

* ReLU
* Sigmoid
* Tanh

---

## 4️⃣ 학습 과정

### ✔ 순전파 (Forward Propagation)

* 입력 → 출력으로 계산

---

### ✔ 손실 함수 (Loss Function)

* 예측값과 실제값 차이 계산

---

### ✔ 역전파 (Backpropagation)

* 오차를 기반으로 가중치 업데이트

---

### ✔ 최적화 (Optimizer)

* 가중치 조정 방법

📌 예:

* SGD
* Adam

---

## 5️⃣ 과적합 방지

### ✔ 과적합 (Overfitting)

> 👉 학습 데이터에만 너무 맞는 상태

### ✔ 해결 방법

* Dropout
* 데이터 증가
* 정규화 (Regularization)

---

## 6️⃣ 주요 딥러닝 모델

* CNN → 이미지 처리
* RNN → 시계열 / 텍스트
* Transformer → NLP, 최신 모델

---

## 7️⃣ 학습 흐름

```
데이터 입력 → 순전파 → 손실 계산 → 역전파 → 가중치 업데이트 → 반복
```
