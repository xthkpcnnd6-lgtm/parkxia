# 🧠 신경망 (Neural Network) 핵심 이론 정리

---

## 1️⃣ 퍼셉트론 (Perceptron)

> 👉 가장 기본적인 신경망 모델 (이진 분류)

* 입력값 × 가중치 + 편향 → 출력
* 결과: 0 또는 1

```text
y = w₁x₁ + w₂x₂ + b
```

✔ 핵심

* 선형 분리 가능 문제 해결
* 단층 구조

---

## 2️⃣ 퍼셉트론 분류 경계

> 👉 데이터를 나누는 직선 (Decision Boundary)

* 직선 하나로 두 그룹 분리
* 선형 모델의 한계 존재

✔ 핵심

* 직선으로 나뉘면 가능
* XOR 같은 문제는 불가능

---

## 3️⃣ 활성화 함수 (Activation Function)

> 👉 출력에 비선형성 추가

* Step Function (계단 함수)
* ReLU, Sigmoid 등으로 확장

✔ 핵심

* 신경망이 복잡한 패턴 학습 가능하게 만듦

---

## 4️⃣ 다층 퍼셉트론 (MLP)

> 👉 퍼셉트론을 여러 층으로 확장

```text
입력층 → 은닉층 → 출력층
```

✔ 특징

* 비선형 문제 해결 가능
* 딥러닝의 기본 구조

---

## 5️⃣ 순전파 (Forward Propagation)

> 👉 입력 → 출력으로 계산 진행

```text
입력 → 가중치 계산 → 활성화 함수 → 출력
```

✔ 핵심

* 예측값을 만드는 과정

---

## 6️⃣ 손실 함수 (Loss Function)

> 👉 예측과 실제값의 차이

```text
Loss = (실제값 - 예측값)^2
```

✔ 핵심

* 값이 작을수록 좋은 모델

---

## 7️⃣ 역전파 (Backpropagation)

> 👉 오차를 기반으로 가중치 수정

```text
오차 → 기울기 계산 → 가중치 업데이트
```

✔ 핵심

* 신경망 학습의 핵심 알고리즘

---

## 8️⃣ 가중치 업데이트 (학습)

> 👉 모델을 점점 개선

```text
w = w - 학습률 × 기울기
```

✔ 핵심

* 반복할수록 성능 향상

---

# 🔥 전체 흐름

```text
입력 → 순전파 → 예측 → 손실 계산 → 역전파 → 가중치 업데이트 → 반복
```
