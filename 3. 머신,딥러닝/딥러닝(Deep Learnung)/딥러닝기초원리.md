# 🚀 모듈 6: 엔드 투 엔드(End-to-End) 신경망 훈련 파이프라인

## 1. 훈련 루프의 5단계 템플릿 (Standard Loop)
모든 PyTorch 학습은 다음 5단계가 반복됩니다. 이 순서를 '외우는 것'이 파이프라인 구축의 핵심입니다.

1.  **Forward:** `predictions = model(x)` (데이터를 넣어 예측값 계산)
2.  **Loss:** `loss = criterion(predictions, y)` (정답과 비교하여 틀린 정도 계산)
3.  **Zero Grad:** `optimizer.zero_grad()` (이전 단계의 기울기 찌꺼기 제거)
4.  **Backward:** `loss.backward()` (오차를 뒤로 전파하며 기울기 계산)
5.  **Step:** `optimizer.step()` (가중치를 업데이트하여 한 걸음 이동)

---

## 2. XOR 문제: 비선형성의 정복
* **의미:** 단층 퍼셉트론으로는 풀 수 없었던 XOR 문제를 다층 구조(Hidden Layer)와 활성화 함수(ReLU)를 통해 스스로 가중치를 찾아내어 해결합니다.
* **핵심:** `nn.Linear` 층 사이의 `torch.relu`가 비선형적인 경계선을 만들어냅니다.

---

## 3. 실전 파이프라인 구성 요소
실제 데이터를 다룰 때는 모델 외에도 다음의 '관리 도구'가 필수입니다.

* **StandardScaler:** 데이터의 단위(cm, mm 등)를 통일하여 학습 속도와 안정성을 높임.
* **DataLoader:** 데이터를 한꺼번에 넣지 않고 쪼개서(Mini-batch) 넣어 메모리를 효율적으로 관리함.
* **Train/Val/Test 분리:** 
  * **Train:** 공부용 문제집
  * **Validation:** 공부 중간에 보는 모의고사 (과적합 방지)
  * **Test:** 최종 수능 시험 (모델의 진짜 실력)

---

## 4. 문제 유형별 최적의 조합 (Golden Rules)

| 문제 유형 | 출력층 활성화 | 손실 함수 (Loss) | 비고 |
| :--- | :--- | :--- | :--- |
| **이진 분류** | `Sigmoid` | `BCELoss` | 0 또는 1 판정 |
| **다중 분류** | `Logits` (없음) | `CrossEntropyLoss` | 내부적으로 Softmax 포함 |
| **회귀** | `Linear` (없음) | `MSELoss` | 연속적인 수치 예측 |

---

## 5. 학습 성과 시각화의 중요성
* **Loss Graph:** 에폭이 지날수록 손실이 우하향하는지 확인.
* **Accuracy Graph:** 훈련(Train)과 검증(Val) 정확도가 함께 올라가는지 확인.
  * *만약 Train만 오르고 Val은 떨어진다면?* → **과적합(Overfitting)** 발생 신호!
