# 🧠 심층 신경망(Deep Neural Network) 핵심 요약 노트

심층 신경망(DNN)은 입력층과 출력층 사이에 여러 개의 **은닉층(Hidden Layers)**을 쌓아 데이터의 복잡한 패턴을 학습하는 인공신경망입니다.

---

## 1. 심층 신경망의 기본 구조

* **입력층 (Input Layer):** 외부 데이터를 모델로 받아들이는 층.
* **은닉층 (Hidden Layer):** 데이터의 특징을 추출하고 학습하는 층 (여러 층일수록 'Deep'해짐).
* **출력층 (Output Layer):** 최종 예측값(분류, 회귀 등)을 내보내는 층.

---

## 2. 신경망의 핵심 동작 원리

### ① 순전파 (Forward Propagation)
입력 데이터에 **가중치(Weight)**를 곱하고 **편향(Bias)**을 더해 활성화 함수를 거쳐 다음 층으로 전달하는 과정입니다.
- 연산: $y = f(W \cdot x + b)$

### ② 손실 함수 (Loss Function)
모델의 예측값과 실제 정답의 차이를 수치화합니다.
- **회귀:** MSE (Mean Squared Error)
- **분류:** Cross-Entropy Loss

### ③ 역전파 (Backpropagation)
출력층에서 발생한 오차를 입력층 방향으로 거슬러 올라가며 전달하여, 각 층의 가중치를 얼마나 수정해야 할지 계산합니다. (**연쇄 법칙(Chain Rule)** 기반)

### ④ 옵티마이저 (Optimizer)
계산된 오차를 바탕으로 가중치를 실제로 업데이트하는 알고리즘입니다.
- 종류: SGD, Adam, RMSprop 등

---

## 3. 필수 구성 모듈 및 개념

### 🔹 활성화 함수 (Activation Function)
선형 연산 결과를 비선형으로 변환하여 모델의 표현력을 높입니다.
- **ReLU:** $f(x) = \max(0, x)$ (가장 널리 사용됨)
- **Sigmoid:** 결과를 0~1 사이로 압축 (이진 분류 출력층)
- **Softmax:** 결과를 확률 분포로 변환 (다중 분류 출력층)

### 🔹 층(Layer)의 종류
- **Linear / Fully Connected Layer:** 모든 노드가 서로 연결된 기본 층.
- **Batch Normalization:** 학습 과정에서 각 층의 입력 분포를 일정하게 맞춰 학습 속도와 안정성을 향상.
- **Dropout:** 학습 시 무작위로 노드를 꺼서 **과적합(Overfitting)**을 방지.

---

## 4. 학습 최적화 테크닉

1.  **Learning Rate (학습률):** 가중치를 한 번에 얼마나 수정할지 결정하는 수치.
2.  **Epoch & Batch Size:**
    - **Epoch:** 전체 데이터를 한 번 다 훑는 횟수.
    - **Batch Size:** 한 번의 가중치 업데이트를 위해 사용하는 데이터의 묶음 단위.
3.  **Early Stopping:** 검증 데이터에 대한 성능이 더 이상 좋아지지 않으면 학습을 중단하여 과적합 방지.

---

## 5. 파이토치(PyTorch) 예시 코드 구조

```python
import torch.nn as nn

class DeepModel(nn.Module):
    def __init__(self):
        super(DeepModel, self).__init__()
        self.network = nn.Sequential(
            nn.Linear(784, 256), # 입력층 -> 은닉층 1
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(256, 128), # 은닉층 1 -> 은닉층 2
            nn.ReLU(),
            nn.Linear(128, 10)   # 은닉층 2 -> 출력층
        )

    def forward(self, x):
        return self.network(x)


# 📉 최적화 핵심: 경사하강법과 연쇄법칙(Chain Rule)

## 1. 기울기(Gradient)의 본질
* **정의:** 손실함수를 가중치($W$)로 미분한 값 ($\frac{\partial L}{\partial W}$).
* **역할:** "가중치를 어느 방향으로, 얼마만큼 바꿔야 손실($L$)이 줄어드는가?"를 알려주는 지표.
* **업데이트 공식:** $W_{new} = W_{old} - \eta \frac{\partial L}{\partial W}$
  * $\eta$ (학습률): 한 번에 이동할 보폭.
  * `-` 부호: 손실이 줄어드는 방향(기울기 반대 방향)으로 가기 위함.

---

## 2. 연쇄법칙(Chain Rule): 오차의 배달부
딥러닝은 여러 층의 함수가 겹쳐진 구조입니다. 출력층의 오차를 저 멀리 있는 앞쪽 가중치까지 전달하려면 **'중간 단계의 미분값들을 모두 곱하는 규칙'**인 연쇄법칙이 필요합니다.

### 🍎 사과 가격 비유
* **최종금액($z$)** = (사과값 $\times$ 개수) $\times$ 소비세
* **사과값이 최종금액에 주는 영향** = (소비세 미분) $\times$ (개수 미분)
* 즉, 거대한 변화량은 **작은 구간의 변화량(로컬 미분)들의 곱**으로 분해할 수 있습니다.

---

## 3. 역전파(Backpropagation)의 실제 계산 (2층 구조 예시)
오차는 뒤($L$)에서부터 앞($w_1$)으로 흐르며 계산됩니다.

### ① 출력층 오차 전파 ($\delta_2$)
1. **손실 미분:** $(\hat{y} - y)$
2. **활성화 함수 미분:** $\hat{y}(1 - \hat{y})$ (시그모이드 기준)
3. **오차 신호 결합:** $\delta_2 = (\hat{y} - y) \cdot \hat{y}(1 - \hat{y})$

### ② 은닉층 오차 전파 ($\delta_1$)
* 앞선 오차 신호($\delta_2$)에 **현재 층의 가중치($w_2$)**와 **활성화 함수 미분**을 곱해 뒤로 전달합니다.
* $\delta_1 = \delta_2 \cdot w_2 \cdot h(1 - h)$

### ③ 최종 기울기(Gradient) 완성
* 각 층의 가중치 기울기는 계산된 오차 신호($\delta$)에 **그 층에 들어온 입력값**을 곱하면 끝납니다.
* **출력층 기울기:** $\
    delta_2 \times h$ (은닉층 출력값)
* **은닉층 기울기:** $\delta_1 \times x$ (최초 입력값)
