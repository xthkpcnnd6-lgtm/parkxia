# 🧠 딥러닝 핵심 이론 총정리 (신경망 ~ 실무 흐름)

---

# 1️⃣ 신경망 구조 이해

## ✔ 퍼셉트론 (Perceptron)

> 👉 가장 기본적인 이진 분류 모델

* 입력 × 가중치 + 편향 → 출력
* 선형 분류 가능

---

## ✔ 다층 퍼셉트론 (MLP)

> 👉 퍼셉트론을 여러 층으로 확장

```text
입력층 → 은닉층 → 출력층
```

* 비선형 문제 해결 가능
* 딥러닝의 기본 구조

---

## ✔ 활성화 함수

> 👉 비선형성 추가 (핵심)

* Sigmoid → 확률 출력
* ReLU → 가장 많이 사용

---

## ✔ Sigmoid & 자동 미분

* Sigmoid: 0~1 사이 출력
* 자동 미분: 기울기를 자동 계산 (딥러닝 핵심 기술)

---

# 2️⃣ 신경망 학습 원리

## ✔ 순전파 (Forward Propagation)

> 👉 입력 → 출력으로 계산

---

## ✔ 손실 함수 (Loss Function)

> 👉 예측값과 실제값 차이

* MSE (회귀)
* BCE (이진 분류)

---

## ✔ BCE (Binary Cross Entropy)

> 👉 이진 분류 전용 손실 함수

```text
Loss = -[y log(p) + (1-y) log(1-p)]
```

---

## ✔ 역전파 (Backpropagation)

> 👉 오차를 기반으로 가중치 수정

* 기울기 계산
* 체인룰 활용

---

## ✔ 가중치 업데이트

> 👉 모델 학습 핵심

```text
w = w - 학습률 × gradient
```

---

## ✔ 은닉층 / 출력층 업데이트

* 출력층 → 직접 오차 계산
* 은닉층 → 출력층 통해 간접 업데이트

---

# 3️⃣ 분류 문제 발전

## ✔ 이진 분류 (Binary Classification)

* 결과: 0 or 1
* Sigmoid + BCE 사용

---

## ✔ 다중 분류 (Multi-class Classification)

* 여러 클래스 분류
* Softmax + CrossEntropy 사용

---

# 4️⃣ 딥러닝 모델 설계

## ✔ 케라스 (Keras)

> 👉 쉽고 빠르게 모델 설계

```python
from tensorflow.keras import Sequential
from tensorflow.keras.layers import Dense

model = Sequential([
    Dense(10, activation='relu'),
    Dense(1, activation='sigmoid')
])
```

---

## ✔ 파이토치 (PyTorch)

> 👉 유연한 딥러닝 프레임워크

```python
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(2, 10),
    nn.ReLU(),
    nn.Linear(10, 1)
)
```

---

# 5️⃣ 딥러닝 학습 전체 흐름

```text
데이터 입력
 → 순전파
 → 손실 계산
 → 역전파
 → 가중치 업데이트
 → 반복 학습
```

# 6️⃣ 모델 평가 & 분석

## ✔ 모델 성능 측정

* Accuracy
* Precision / Recall

---

## ✔ 결정 계수 (R²)

> 👉 회귀 모델 평가

* 1에 가까울수록 좋음

---

## ✔ 모델 시각화

* 학습 곡선 (loss, accuracy)
* 결과 그래프

---

# 7️⃣ 실무 모델 확장

## ✔ Wide & Deep Model

> 👉 선형 + 딥러닝 결합

* Wide → 단순 패턴
* Deep → 복잡한 패턴

---

## ✔ 학습 → 결과 전체 과정

```text
데이터 → 모델 설계 → 학습 → 평가 → 배포
```

---

# 8️⃣ 과적합 (Overfitting) 방지

## ✔ 과적합

> 👉 학습 데이터에만 맞는 문제

---

## ✔ 해결 방법

* Dropout
* 데이터 증가
* 정규화 (L1, L2)
* Early Stopping
