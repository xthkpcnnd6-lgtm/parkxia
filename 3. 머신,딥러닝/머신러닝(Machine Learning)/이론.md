# 🤖 머신러닝 (Machine Learning) 핵심 정리

---

## 1️⃣ 머신러닝이란?

> 👉 데이터를 통해 **패턴을 학습하고 예측하는 알고리즘**

* 명시적인 규칙 없이 스스로 학습
* 데이터 기반 의사결정

---

## 2️⃣ 머신러닝 종류

### ✔ 지도학습 (Supervised Learning)

> 👉 정답(label)이 있는 데이터로 학습

* 분류 (Classification) → 카테고리 예측
* 회귀 (Regression) → 숫자 값 예측

📌 예:

* 스팸 메일 분류
* 집값 예측

---

### ✔ 비지도학습 (Unsupervised Learning)

> 👉 정답 없이 데이터 구조 파악

* 군집화 (Clustering)
* 차원 축소 (Dimensionality Reduction)

📌 예:

* 고객 그룹 나누기

---

### ✔ 강화학습 (Reinforcement Learning)

> 👉 보상을 통해 학습

* 행동 → 보상 → 최적 전략 학습

📌 예:

* 게임 AI

---

## 3️⃣ 머신러닝 기본 흐름

```text id="vm9d88"
데이터 수집 → 데이터 전처리 → 모델 선택 → 학습 → 평가 → 예측
```

---

## 4️⃣ 핵심 개념

### ✔ 데이터 (Data)

* 학습의 기반
* Feature(입력) + Label(정답)

---

### ✔ 모델 (Model)

* 데이터를 학습하는 알고리즘
* 예: SVM, Decision Tree, Neural Network

---

### ✔ 학습 (Training)

* 데이터로 모델을 훈련시키는 과정

---

### ✔ 평가 (Evaluation)

* 모델 성능 측정

📌 주요 지표:

* 정확도 (Accuracy)
* 정밀도 (Precision)
* 재현율 (Recall)
* F1-score

---

## 5️⃣ 과적합 & 일반화

### ✔ 과적합 (Overfitting)

> 👉 학습 데이터에만 너무 잘 맞음

* 새로운 데이터에서 성능 ↓

### ✔ 해결 방법

* 데이터 증가
* 규제 (Regularization)
* 단순한 모델 사용

---

## 6️⃣ 주요 알고리즘

* 선형 회귀 (Linear Regression)
* 로지스틱 회귀 (Logistic Regression)
* SVM (Support Vector Machine)
* KNN (K-Nearest Neighbors)
* 결정 트리 (Decision Tree)

---
