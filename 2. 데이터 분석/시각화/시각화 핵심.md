# 📊 데이터 시각화 핵심 정리 (Matplotlib & Seaborn)

---

## 1️⃣ 시각화의 목적

> 👉 데이터를 **한눈에 이해하고 패턴을 찾기 위함**

* 추세 파악
* 분포 확인
* 관계 분석
* 비교 분석

---

## 2️⃣ 기본 흐름

> 👉 **데이터 → 그래프 선택 → 시각화 → 해석**

```python
import matplotlib.pyplot as plt

plt.plot(x, y)
plt.title("Title")
plt.xlabel("X")
plt.ylabel("Y")
plt.show()
```

---

## 3️⃣ 주요 그래프 종류

### ✔ 선 그래프 (Line Plot)

* 시간 흐름 / 추세 확인

```python
plt.plot(x, y)
```

### ✔ 막대 그래프 (Bar Plot)

* 카테고리 비교

```python
plt.bar(labels, values)
```

### ✔ 히스토그램 (Histogram)

* 데이터 분포 확인

```python
plt.hist(data)
```

### ✔ 산점도 (Scatter Plot)

* 변수 간 관계 분석

```python
plt.scatter(x, y)
```

### ✔ 박스플롯 (Box Plot)

* 분포 + 이상치 확인

```python
plt.boxplot(data)
```

### ✔ 히트맵 (Seaborn)

* 상관관계 분석

```python
import seaborn as sns
sns.heatmap(data, annot=True)
```

---

## 4️⃣ 그래프 꾸미기 (가독성 핵심)

```python
plt.title("제목")
plt.xlabel("X축")
plt.ylabel("Y축")
plt.legend()
plt.grid()
```

* 제목 필수
* 축 라벨 필수
* 범례(legend) 추가
* grid로 가독성 향상

---

## 5️⃣ 여러 그래프 비교 (Subplot)

```python
plt.subplot(1, 2, 1)
plt.plot(x, y1)

plt.subplot(1, 2, 2)
plt.plot(x, y2)

plt.show()
```

👉 여러 데이터를 동시에 비교 가능

---

## 6️⃣ Seaborn 활용 (고급 시각화)

```python
import seaborn as sns

sns.histplot(data)
sns.boxplot(x='group', y='value', data=df)
sns.heatmap(df.corr(), annot=True)
```

* 통계 기반 시각화
* 자동으로 예쁜 스타일
* pandas와 잘 어울림
