# 📊 결측치 vs 이상치 핵심 개념 정리

---

## 1️⃣ 결측치 (Missing Values)

> 값이 **아예 없는 상태**

### ✔ 정의

* 데이터가 비어있음 (`NaN`, `None` 등)

### ✔ 발생 원인

* 입력 누락
* 측정 실패
* 데이터 수집 오류

### ✔ 확인 방법

* `isna()`
* `isnull()`

### ✔ 처리 방법

* 삭제 → `dropna()`
* 대체 → `fillna()`
* 보간 → `interpolate()`

### 🔥 핵심 포인트

> 👉 **없기 때문에 채우거나 제거해야 함**

---

## 2️⃣ 이상치 (Outliers)

> 다른 값들과 **너무 동떨어진 값**

### ✔ 정의

* 데이터 분포에서 튀는 값

### ✔ 발생 원인

* 입력 오류 (예: `180cm → 1800cm`)
* 특이한 실제 데이터
* 센서 오류

### ✔ 탐지 방법

* IQR (사분위수)
* Z-score
* 시각화 (박스플롯)

### ✔ 처리 방법

* 제거
* 값 대체
* 변환 (로그 등)
* 유지 (의미 있는 경우)

### 🔥 핵심 포인트

> 👉 **무조건 제거 ❌ / 상황 판단 필수**

---

## 📌 한눈에 비교

| 구분    | 결측치     | 이상치       |
| ----- | ------- | --------- |
| 의미    | 값 없음    | 값이 튐      |
| 존재 여부 | ❌ 없음    | ✅ 있음      |
| 처리 방식 | 채우거나 삭제 | 상황에 따라 판단 |

---

## 💡 한 줄 요약

* 결측치 → **없음 → 채움/삭제**
* 이상치 → **이상함 → 판단 후 처리**
