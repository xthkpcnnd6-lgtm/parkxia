# 🤖 SVM (Support Vector Machine) 핵심 정리

---

## 1️⃣ SVM이란?

> 👉 데이터를 **가장 잘 나누는 경계(Decision Boundary)**를 찾는 알고리즘

* 분류(Classification) & 회귀(Regression) 모두 가능
* 지도학습(Supervised Learning) 알고리즘

---

## 2️⃣ 핵심 개념

### ✔ 결정 경계 (Decision Boundary)

* 데이터를 두 그룹으로 나누는 기준선
* SVM은 **최적의 경계**를 찾는 것이 목표

---

### ✔ 마진 (Margin)

> 👉 경계와 데이터 사이의 거리

* 마진이 **클수록 일반화 성능 ↑**
* SVM은 **마진을 최대화**하는 방향으로 학습

---

### ✔ 서포트 벡터 (Support Vector)

> 👉 경계를 결정하는 가장 중요한 데이터

* 경계에 가장 가까운 데이터 포인트
* 이 데이터들만으로 모델이 결정됨

---

### ✔ 커널 트릭 (Kernel Trick)

> 👉 선형 분리가 어려운 데이터를 **고차원으로 변환**해서 해결

* Linear → 직선
* Polynomial → 곡선
* RBF → 가장 많이 사용

---

### ✔ C 값 (규제 파라미터)

> 👉 오차 허용 정도 조절

| C 값 | 의미            |
| --- | ------------- |
| 작음  | 마진 넓음, 단순 모델  |
| 큼   | 마진 좁음, 과적합 가능 |

---

## 3️⃣ SVM 종류

```python id="vrs4lq"
from sklearn.svm import SVC
model = SVC()
```

### ✔ SVR (회귀)

```python id="1d0i7k"
from sklearn.svm import SVR
model = SVR()
```

---

## 4️⃣ 장점 & 단점

### ✅ 장점

* 고차원 데이터에 강함
* 과적합에 비교적 강함
* 커널로 복잡한 문제 해결 가능

### ❌ 단점

* 데이터 많으면 느림
* 파라미터 튜닝 필요
* 해석이 직관적이지 않음

---

## 5️⃣ 핵심 흐름

> 👉 **데이터 → 경계 설정 → 마진 최대화 → 최적 모델 생성**
