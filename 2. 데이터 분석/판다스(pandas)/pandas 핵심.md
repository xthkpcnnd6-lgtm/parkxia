# 📑 판다스 Series 핵심 요약

---

## 1️⃣ 시리즈(Series)의 정체

> **값(Value) + 이름표(Index)**가 붙은 1차원 배열
> 👉 엑셀의 **열(Column) 하나**라고 생각하면 쉬움

### ✔ 구조

* **Index** → 데이터의 주소(라벨)
* **Values** → 실제 데이터

### ✔ 특징

* 넘파이 기반 → 연산 속도 빠름
* 인덱스 이름으로 데이터 접근 가능

---

## 2️⃣ 데이터 만들고 가져오기

### ✔ 생성

```python
import pandas as pd

s = pd.Series([10, 20], index=['a', 'b'])
```

### ✔ 데이터 접근 방법

* 레이블 인덱싱 → `s['a']`
* 위치 인덱싱 → `s[0]`
* 조건 인덱싱 → `s[s > 15]`

---

## 3️⃣ 마법 같은 연산과 통계

### ✔ 벡터화 연산

```python
s * 2  # 모든 값이 2배
```

### ✔ 통계 요약

```python
s.describe()
```

* 평균, 표준편차, 최댓값 등 자동 계산

### ✔ 자동 정렬 연산

* 서로 다른 Series 연산 시
  👉 **같은 인덱스끼리 자동 매칭**

---

## 4️⃣ 데이터 정제 (결측치 & 정렬)

### ✔ 결측치 처리

```python
s.dropna()      # 제거
s.fillna(값)     # 대체
```

### ✔ 정렬

```python
s.sort_values()  # 값 기준 정렬
s.sort_index()   # 인덱스 기준 정렬
```

---

## 📌 한눈에 핵심 정리

* Series = **1차원 + Index**
* 인덱스로 자유롭게 데이터 접근 가능
* 연산은 **반복문 없이 한 줄**
* 결측치 & 정렬은 데이터 정제의 핵심
