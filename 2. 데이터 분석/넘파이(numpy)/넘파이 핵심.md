# 📦 NumPy 핵심 요약

---

## 1️⃣ 배열 생성 및 초기화

### ✔ 배수 생성

```python
import numpy as np

np.arange(3, 101, 3)  # 1~100 사이 3의 배수
```

### ✔ 특수 행렬 (초기화)

```python
np.zeros((5, 5))  # 0으로 채워진 5x5 행렬
```

* 슬라이싱을 활용하면 **체크보드 패턴 등 다양한 구조 생성 가능**

### ✔ 난수(Random)

```python
np.random.rand(3)            # 0~1 균등분포
np.random.randn(3)           # 표준정규분포 (평균 0, 표준편차 1)
np.random.normal(10, 2, 3)   # 평균 10, 표준편차 2
```

## 2️⃣ 데이터 접근 및 필터링 (핵심)

### ✔ 불리언 인덱싱

```python
arr[arr > 5]
arr[(arr > 3) & (arr < 8)]
```

### ✔ 팬시 인덱싱

```python
arr[[0, 2, 5]]  # 원하는 위치만 선택
```

### ✔ 슬라이싱

```python
arr[1:5:2]  # 시작:끝:간격
```

* 다차원 배열에서도 동일하게 적용 가능

---

## 3️⃣ 넘파이의 동작 원리

### ✔ View vs Copy

```python
view = arr[:3]      # View (원본 공유)
copy = arr[:3].copy()  # Copy (독립)
```

* View 수정 시 → 원본도 변경됨
* 원본 보호하려면 `.copy()` 필수

---

### ✔ 브로드캐스팅 (Broadcasting)

```python
arr = np.zeros((3, 4))
arr + np.array([1, 2, 3, 4])
```

* 서로 다른 shape도 자동으로 맞춰 연산

📌 예시:

* `(3, 4)` + `(4,)`
* `(3, 4)` + `(3, 1)`

---

## 4️⃣ 데이터 변형

### ✔ Reshape

```python
arr.reshape(2, 2)
arr.reshape(2, -1)  # 자동 계산
```

### ✔ 차원 개념 (Dimension)

* 0차원 → 스칼라
* 1차원 → 벡터
* 2차원 → 행렬
* 3차원 → 텐서

---

## 📌 한눈에 핵심 정리

* NumPy = **빠른 배열 연산 라이브러리**
* 인덱싱 → 데이터 선택의 핵심
* 브로드캐스팅 → 연산 자동화 핵심
* reshape → 데이터 구조 변경 핵심
